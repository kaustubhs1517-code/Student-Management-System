# Student Management System

A beginner-friendly Student Management System built using Python.
The project stores student data permanently using a JSON file and supports CRUD operations.

## Features

## Features

- ➕ Add Student
- 👀 View Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- 🗑️ Delete Student
- 💾 Store Data in JSON File
- 🔄 Automatic Data Loading on Startup
- ✅ Prevent Duplicate Roll Numbers
- 📝 Input Validation
  - Student name cannot be empty
  - Student name accepts only letters and spaces
  - Roll number cannot be empty
  - Roll number accepts only digits
  - Course cannot be empty
- 🎨 Attractive Welcome Banner
- 📋 User-friendly Menu Interface

## Technologies Used

- Python
- JSON
- Git
- GitHub
